**Weather App** is a live weather forecasting.

## Features

* Shows live Weather forecast for every city worldwide.
* Attractive User Interface
* Displayed info:
  * Temperature in Celcius
  * Windspeed 
  * Longitude Latitude 
  * Pressure and Humidity
  
## API Used 

* [OpenWeather](https://openweathermap.org/)

## Technology Used

* HTML/CSS
* EJS
* Node.Js
